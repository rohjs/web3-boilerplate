# web3-boilerplate w/ wagmi

## Requisition

- TypeScript v4.x
- Node v18.12.0 (recommend to use LTS version)

## Run dev server

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
# or
yarn build
```
